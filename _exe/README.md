#### Remark
```
 --------------------------------------------------
 There are default values for these parameters:

 reductionPercentage = 0.0
 maxWidth  = 5000 -> 50000px
 maxHeight = 5000 -> 50000px
 chars     = "@%#*+=:~-. "

 --------------------------------------------------
 Default values can be activated by specifying:

 reductionPercentage < 0 || reductionPercentage > 1
 maxWidth  <= 0
 maxHeight <= 0
 chars     == ""
 --------------------------------------------------
```
